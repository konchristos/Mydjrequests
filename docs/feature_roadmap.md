# Feature Roadmap -- MyDJRequests

## Phase 1

-   Rekordbox library importer
-   Song request system
-   Library matching
-   Playlist generator

## Phase 2

-   Missing track report
-   Spotify metadata enrichment
-   Smart track suggestions

## Phase 3

-   BPM and key based playlist generation
-   Energy curve playlist builder
-   Automated DJ set preparation tools

## Long Term Vision

Transform MyDJRequests from a request platform into a DJ event
preparation assistant.
